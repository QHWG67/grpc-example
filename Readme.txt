# Project Description
This project demonstrates a gRPC-based client-server application in C++. It includes a server that provides user information and a client that retrieves this information using gRPC.

# 项目描述
这个项目展示了一个基于 gRPC 的 C++ 客户端-服务器应用程序。它包括一个提供用户信息的服务器和一个使用 gRPC 检索信息的客户端。

# How to Build and Run
1. **Install Dependencies**:
   - Ensure you have gRPC and Protocol Buffers installed.
   - Install CMake (version 3.25 or higher).

2. **Build the Project**:
   ```bash
   mkdir build
   cd build
   cmake ..
   make
   ```

3. **Run the Server**:
   ```bash
   ./server
   ```

4. **Run the Client**:
   ```bash
   ./client
   ```
